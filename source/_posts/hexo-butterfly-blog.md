## 为什么选择 Hexo？

Hexo 是一个快速、简洁且高效的静态博客框架。它使用 Markdown 解析文章，在几秒内即可生成静态页面。

### 优势

- **速度快**：Node.js 驱动，生成速度极快
- **生态丰富**：数百个主题和插件可选
- **部署简单**：一键部署到 GitHub Pages

## Butterfly 主题

Butterfly 是 Hexo 生态中最受欢迎的主题之一，拥有：

1. 卡片式文章布局
2. 内置暗色模式
3. 丰富的侧边栏组件
4. 多种背景特效

```javascript
// 安装 Butterfly
npm install hexo-theme-butterfly
```

## 总结

搭建博客并不难，难的是坚持写作。希望这个博客能成为记录学习和成长的地方。

> 好的开始是成功的一半。
